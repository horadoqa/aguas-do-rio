# aguas-do-rio

Comunicados

O Código:

- Ele acessa a URL: https://aguasdorio.com.br/comunicados/
- Ele coleta todos os <h5 class="card-title"> da página.
- Para cada <h5> encontrado, ele imprime na tela:
    - Encontrado na página: Rio Bonito
    - Email enviado com sucesso!
- Se "Rio Bonito" não aparecer, ele apenas mostrará no console:
    - Palavra não encontrada hoje.
